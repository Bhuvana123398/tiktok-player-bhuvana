# TikTok Video Player Clone

##  Project Description
This project is a TikTok-style video player built using React. It allows users to scroll through videos with autoplay functionality and interact using like, comment, share, save, follow, and music controls.

##  Features
- 🎥 Auto-play videos on scroll
- 🔊 Global mute/unmute functionality
- ❤️ Like button (increments count)
- 💬 Comment button (increments count)
- 🔗 Share button (increments count)
- 🔖 Save button (toggle + count)
- 👤 Follow/Following toggle button
- 🎵 Music icon rotates when sound is ON
- 🔁 Infinite scrolling
##  Tech Stack
- React.js

- HTML5 & CSS3
